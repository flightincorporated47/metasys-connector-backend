\# CSUDH Pilot — Connector SOP Index



Use these SOPs in order for the pilot:



1\. \*\*SOP-CONN-005 — Metasys API Key Handling\*\*

&nbsp;  - Set up secrets safely (no keys in YAML or git)



2\. \*\*SOP-CONN-007 — Network/VLAN Access Request Template\*\*

&nbsp;  - Request approved ranges/ports from CSUDH IT (read-only + outbound-only)



3\. \*\*SOP-CONN-001 — BACnet Scanner Run\*\*

&nbsp;  - Discover devices safely and validate output



4\. \*\*SOP-CONN-006 — Pilot Point Selection Criteria\*\*

&nbsp;  - Choose 100–300 points and assign tiering rules



5\. \*\*SOP-CONN-003 — Points Import + Tier Assignment\*\*

&nbsp;  - Import CSV and apply tiers/deadbands



6\. \*\*SOP-CONN-002 — Metasys Connector Deploy (Read-only)\*\*

&nbsp;  - Run locally first, validate stability/traffic



7\. \*\*SOP-CONN-004 — Publishing Enablement Gate\*\*

&nbsp;  - Only enable publishing after approval + queue/replay testing
